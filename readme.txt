Allows to Create login and logout log file.

== Description ==

This plugin allows to create login and logout log file. Log file stores login or logout date and time, user_name, user_role, remote_ip, and Event. Log file created on given the file location.

PLUGIN FUNCTIONALITY:
1. Avails all login and logout .
 
== Installation ==
1. Ensure you have latest version of the plugin zip.
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

== Important Notes ==
1. Set the log file path from the admin dashboard in Audit log viewer 
2. The web host path should be writable.
